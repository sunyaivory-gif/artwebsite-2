# Art Website Vercel Deploy Folder

This folder is the public deploy copy of the art website.

## Files

- `index.html` - the website file Vercel should publish.

## Local Editing Workflow

Edit the original site here:

`C:\Users\Admin\working\ai work\artwebsite_2.html`

When the local site looks right, copy it into this folder as `index.html`, then upload or push this folder to GitHub for Vercel.

## Vercel Settings

- Framework preset: Other / Static
- Build command: leave blank
- Output directory: leave blank
- Root directory: this folder, if it is inside a larger repo
